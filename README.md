## Dangling-else 的解决方法

```
Stmt ::= if Exp Stmt [else Stmt] | ...
```

转化为

```
Stmt ::= OpenStmt | ClosedStmt
ClosedStmt ::= if Exp ClosedStmt else ClosedStmt | BasicStmt
OpenStmt ::= if Exp Stmt | if Exp ClosedStmt else OpenStmt
BasicStmt ::= Resturn | Assgn | Exp | Block
```
